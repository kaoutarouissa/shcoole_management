# School Management (Simple PHP Project)

This is a basic PHP + MySQL project with:

- User registration
- User login

## Files

- `index.php` -> Login form
- `login.php` -> Login processing
- `registerForm.php` -> Register form
- `register_post.php` -> Register processing
- `connection.php` -> Database connection
- `welcome.php` -> Simple success page

## Requirements

- XAMPP (server)
- PHP
